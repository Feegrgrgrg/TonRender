Инструкция к запуску Бота:

1. Откройте файл config.py и введите следующие данные:
    CHANNEL_USERNAME = '@'  # Юзернейм ТГ канала 
    TOKEN = ""  # Токен бота в телеграм
    PAYMENT_TOKEN = '' #Токен платежки

2. В файле pay.py измени сеть TEST_NET на MAIN_NET 

3. Запусти файл main.py

Команды:
/draws - кол-во отрисовок пользователя
/admin - убрать отрисовки у пользователя



Версии:
3.11.9 версия питона,
2.21 версия aiogram
9.5.0 версия pillow
